# Egosy Treat Enterprises - Order App

Welcome to the premium ordering application for **Egosy Treat Enterprises**, your one-stop shop for handcrafted pastries and bakery products in Port-harcourt, Nigeria.

## 🍩 About Egosy Treat Enterprises
We specialize in high-quality, freshly baked snacks and pastries including:
- **Snacks**: Chinchin, Roasted Peanuts, Puff puff, Buns.
- **Pastries**: Meat pie, Egg Roll, Doughnuts.
- **Bakery**: Bread, assorted rolls, and customized treat packs.

## 🚀 Features
- **Vibrant Modern UI**: A premium, mobile-responsive design with glassmorphism and smooth animations.
- **Product Filtering**: Browse treats by category (Snacks, Pastries, Bakery).
- **Interactive Cart**: Add/remove items and adjust quantities with a real-time subtotal.
- **WhatsApp Integration**: Fast checkout by sending your order directly to our WhatsApp.
- **Login Portal**: Professional login page for returning customers.
